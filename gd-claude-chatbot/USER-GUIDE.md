## Quick Start

### Finding the Chatbot

The chatbot can appear in two ways:

#### **Floating Widget** (Default)

The chatbot appears as a **floating button** in the corner of your screen:

- **Look for**: A circular button with a chat icon
- **Location**: Bottom-right corner (configurable)
- **Color**: Customizable (default: red)
- **Animation**: Gentle pulsing glow

#### **Inline with Shortcode** (NEW in v1.8.3)

You can also embed the chatbot directly in any page or post using the shortcode:

```
[gd_chatbot]
```

**Custom Options:**
```
[gd_chatbot title="Ask Me About the Dead" width="500" height="700" color="#DC143C"]
```

**Shortcode Parameters:**
- `title` - Custom chat title
- `welcome` - Custom welcome message
- `width` - Chat width in pixels (default: 420)
- `height` - Chat height in pixels (default: 650)
- `color` - Primary color (default: #DC143C)

### Starting a Conversation

1. **Click the chat button** to open the chat window (or it's already open if using shortcode)
2. **Type your question** in the input box at the bottom
3. **Press Enter or click the send button** (arrow icon)
4. **Watch the response stream in** word by word with a blinking cursor (‚ñã)

That's it! The chatbot is ready to help.

## Troubleshooting

### Chatbot Won't Open

**Problem:** Clicking skull button doesn't open chat

**Solutions:**
- Refresh the page
- Clear browser cache
- Check if JavaScript is enabled
- Try a different browser

### No Response

**Problem:** Sent message but no response

**Solutions:**
- Check your internet connection
- Look for error message
- Wait a moment (might be processing)
- Try resending

### Error Messages

**"Something went wrong"**
- Temporary issue
- Click send again
- Usually resolves quickly

**"Security check failed"**
- Refresh the page
- Clear cookies
- Contact site admin if persists

### Slow Responses

**Normal wait times:**
- First text: 1-3 seconds
- Complex questions: Slightly longer
- Streaming should start quickly

**If very slow:**
- Check internet speed
- Server might be busy
- Try simpler question first

### Text Not Streaming

**Problem:** Response appears all at once

**This is normal if:**
- Response is very short
- Browser doesn't support streaming
- Connection issue occurred

**Still works fine, just different display**

### Formatting Issues

**If text looks strange:**
- Refresh page
- Check browser compatibility
- May be markdown rendering issue
