## ğŸ”§ Troubleshooting

### Don't See New Theme?

1. **Clear Browser Cache**
   - Chrome/Edge: Ctrl+Shift+Delete
   - Firefox: Ctrl+Shift+Delete
   - Safari: Cmd+Option+E
   - Or just hard refresh: Ctrl+F5 (Cmd+Shift+R on Mac)

2. **Clear WordPress Cache**
   - If using WP Super Cache, W3 Total Cache, etc.
   - Clear all caches

3. **Verify Version**
   - Go to Plugins page
   - Should show "2.0.6"

### Still See Old Theme?

Check browser console (F12) for:
- CSS loading errors
- 404 errors for gd-theme.css

If `gd-theme.css` isn't loading:
- Verify file exists at `/wp-content/plugins/gd-chatbot/public/css/gd-theme.css`
- Check file permissions
