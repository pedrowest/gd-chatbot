üéµ Optional: Connect Streaming Services

For Administrators

Configure API credentials for streaming services:

  1. Go to GD Chatbot v2 ‚Üí Settings ‚Üí Streaming Services
  2. Create developer accounts on desired services
  3. Enter Client ID and Client Secret for each service
  4. Add the Redirect URI to each service's app settings
  5. Click "Test Connection" to verify
  6. Click "Save All Configurations"

Claude Settings

  • Model: claude-sonnet-4-20250514 (recommended)
  • Temperature: 0.7 (balanced creativity)
  • Max Tokens: 4096 (standard responses)
  • System Prompt: Customize the chatbot's personality

Music Streaming Settings

  • Enable Streaming: Turn music links on/off
  • Default Sort: Most Popular, Date, or Rating
  • Result Limit: 10-100 performances per song
  • Cache Duration: 1-168 hours

Tavily Settings (Optional)

  • Enable Search: Turn web search on/off
  • Search Depth: Basic (faster) or Advanced (thorough)
  • Max Results: 3-10 results per search
  • Domain Filtering: Use trusted domains list

üîß Troubleshooting

Song Links Not Appearing?

  • Go to Settings ‚Üí Music Streaming
  • Verify "Enable Music Streaming" is checked
  • Click "Clear Song Cache"
  • Refresh the page

Modal Opens But No Results?

  • Go to Music Streaming dashboard
  • Check "Database Status" shows 2,340+ shows
  • If empty, click "Sync All Years"
  • Wait for sync to complete

Chatbot Not Responding?

  • Verify Claude API key is correct
  • Check API key has sufficient credits
  • Look for errors in browser console (F12)
  • Check WordPress error logs
