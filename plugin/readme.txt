== Description ==

GD Chatbot is a powerful WordPress plugin that brings intelligent conversational AI to your website using Anthropic's Claude AI. With advanced features including real-time streaming responses, Tavily web search integration, and Pinecone vector database support, GD Chatbot delivers context-aware, intelligent conversations that engage your visitors.

**Key Features:**

* **Claude AI Integration** - Powered by Anthropic's latest Claude models for natural, intelligent conversations
* **Real-time Streaming** - Instant response streaming for immediate feedback and better user experience
* **Tavily Web Search** - Real-time web search capabilities for up-to-date information
* **Pinecone Vector Database** - Semantic search and RAG (Retrieval Augmented Generation) for custom knowledge bases
* **Knowledge Base Integration** - Supports WordPress Knowledgebase Loader and AI Power plugins
* **CSV Setlist Search** - Built-in database search functionality
* **Customizable Themes** - Professional and psychedelic theme options
* **Responsive Design** - Beautiful chat widget that works perfectly on desktop and mobile
* **Flexible Display Options** - Embed with shortcodes, floating widget, or inline placement
* **Conversation History** - Automatic chat session tracking and restoration
* **Advanced Customization** - Full control over colors, positioning, and appearance
* **Analytics Dashboard** - Track conversations, usage, and engagement metrics
* **Secure by Design** - Encrypted API keys, nonce verification, and capability checks

**Perfect For:**

* Websites needing intelligent customer support
* Content sites requiring AI-powered assistance
* Communities wanting engaging conversational experiences
* Businesses seeking to enhance user engagement
* Developers building custom knowledge-base chatbots

**How It Works:**

GD Chatbot seamlessly integrates with your WordPress site, providing visitors with an intelligent chat assistant that can answer questions, search the web in real-time, and retrieve information from your custom knowledge base. The streaming response feature ensures visitors see answers appear in real-time, creating a responsive and engaging experience.

== Installation ==

1. Upload the `gd-chatbot` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings â†’ GD Chatbot in the admin menu
4. Enter your Claude API key (required) from Anthropic
5. Optionally configure Tavily API key for web search
6. Optionally configure Pinecone credentials for vector database
7. Customize the chat widget appearance and behavior
8. Add the chatbot to your site using the shortcode `[gd_chatbot]` or enable the floating widget

**Minimum Requirements:**

* WordPress 5.0 or higher
* PHP 7.4 or higher
* Active Anthropic Claude API key (get one at https://anthropic.com)
* Optional: Tavily API key for web search (https://tavily.com)
* Optional: Pinecone API credentials for vector database (https://pinecone.io)

== Frequently Asked Questions ==

= Do I need a Claude API key? =

Yes, an active Anthropic Claude API key is required. You can sign up for one at https://anthropic.com.

= What Claude models are supported? =

GD Chatbot supports all current Claude models, with claude-sonnet-4-20250514 recommended as the default.

= Can I use my own knowledge base? =

Yes! GD Chatbot integrates with Pinecone vector database for custom knowledge bases, as well as WordPress Knowledgebase Loader and AI Power plugins.

= Is the Tavily web search required? =

No, Tavily web search is optional. The chatbot works perfectly with just Claude AI. Tavily adds real-time web search capabilities for up-to-date information.

= Can I customize the appearance? =

Absolutely! GD Chatbot offers extensive customization options including colors, themes, positioning, and sizing through the admin settings panel.

= Does it work on mobile devices? =

Yes, the chatbot is fully responsive and works beautifully on all devices, from smartphones to desktop computers.

= How does streaming work? =

Streaming displays Claude's responses in real-time as they're generated, providing immediate feedback and a better user experience compared to waiting for complete responses.
